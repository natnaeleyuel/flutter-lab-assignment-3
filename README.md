# Flutter Lab Assignment 3

A mobile application development assignment built with **Flutter** and **Dart**, representing a transition from earlier native Android assignments into cross-platform app development. The project is structured as a full Flutter application and, based on repository metadata, appears to implement an album-list app with offline support.

## Project Overview

This assignment introduces Flutter as a modern framework for building applications across multiple platforms from a shared codebase. Unlike the earlier Android-only assignments, this project expands into cross-platform development with support for mobile, web, and desktop targets.

## Contents

- **`lib/`**: Main Dart application source code.
- **`test/`**: Test files for validating app behavior.
- **`android/`**: Android platform configuration and native integration files.
- **`ios/`**: iOS platform files.
- **`web/`**: Web build support.
- **`linux/`**, **`macos/`**, **`windows/`**: Desktop platform support files.
- **`pubspec.yaml`**: Flutter dependencies and project metadata.
- **`pubspec.lock`**: Locked dependency versions.
- **`analysis_options.yaml`**: Dart and Flutter linting configuration.
- **`README.md`**: Assignment details and project overview.

## Features

- Built with **Flutter** for cross-platform application development.
- Uses **Dart** as the primary application language.
- Organized as a complete multi-platform Flutter project.
- Repository history suggests an **album list application** with **offline support**.
- Suitable as a practical step forward from native Android development into shared-code mobile engineering.

## Usage

1. Clone the repository:
   ```sh
   git clone https://github.com/natnaeleyuel/flutter-lab-assignment-3.git
   ```
2. Open the project in **VS Code** or **Android Studio** with Flutter support installed.
3. Install dependencies:
   ```sh
   flutter pub get
   ```
4. Run the app on your preferred target:
   ```sh
   flutter run
   ```
